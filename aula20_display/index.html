1. Display: Block

Elementos com display: block ocupa toda a largura disponível, criando uma nova linha anates e depois do elemento.

Este é um bloco. Ele ocupa toda a largura disponível, criando uma nova quebra de linha.
Outro bloco, o que mostra que os blocos vão para a próxima linha.

2. Display: Inline

Elesmentos com display: inline se alinham horizontalmente, respeitando apenas o seu conteúdo e não criam quebras de linha.

Este é um elemento inline. Outro elemento inline. Mais um elemento inline.

3. Display: Inline-Block

Elementos com display: inline-block se alinham horizontalmente, mas podem ter largura e altura definidas, ao contrário do inline.

Este é outro elemento inline-block.
Outro elemento inline-block.
Mais um elemento inline-block.
